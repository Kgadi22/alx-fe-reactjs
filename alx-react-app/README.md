# ALX React App

This project is a simple React application created with Vite for learning and practicing React concepts, including JSX and component composition.

## Features
- Welcome message component
- Header, Main Content, and Footer components
- Easy to extend and experiment with React features

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open your browser and go to [http://localhost:5173](http://localhost:5173)

## Project Structure
- `src/components/` - Contains all React components
- `src/App.jsx` - Main application component
